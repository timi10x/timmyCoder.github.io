# How to Run Portfolio-v2 with Professional App Showcase

## Prerequisites
You need Node.js installed on your system. 

### Install Node.js:
1. Visit https://nodejs.org/
2. Download and install the LTS version for macOS
3. After installation, restart your terminal

## Running the Portfolio

1. Open Terminal
2. Navigate to the portfolio-v2 directory:
   ```bash
   cd "/Users/timi10x/Desktop/my portfolio/timi10x.com/portfolio-v2"
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Run the development server:
   ```bash
   npm run dev
   ```

5. Open your browser and go to: http://localhost:3000

6. Click on the Footlocker project to see the professional app showcase!
